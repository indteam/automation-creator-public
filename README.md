# Automation Creator Add-on

AI-powered automation creation for Home Assistant using natural language commands.

## Installation

1. Add this repository to Home Assistant:
   - Go to **Settings** → **Add-ons** → **Add-on Store**
   - Click the three dots (⋮) in the top right
   - Select **Repositories**
   - Add: `https://github.com/indteam/automation-creator-public`

2. Install the add-on:
   - Find "Automation Creator" in the Add-on Store
   - Click **Install**
   - Configure your OpenAI API key
   - Start the add-on

## Configuration

- **OpenAI API Key**: Your OpenAI API key (required)

## Usage

Once installed, you can create automations via:
- Voice commands through Assist
- HTTP API calls
- Webhook integration
